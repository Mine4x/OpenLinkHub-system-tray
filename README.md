# OpenLinkHub-system-tray

OpenLinkHub-system-tray is an application that lets you display info
like your headphones battery on the system tray.

I made this just for fun and personal use.

## Dependencies

### libayatana-appindicator

Arch based systems:
```bash
sudo pacman -S libayatana-appindicator
```

Debian based systems:
```bash
sudo apt install libayatana-appindicator3-dev
```

Fedora:
```bash
sudo dnf install libayatana-appindicator-gtk3-devel
```

openSUSE:
```bash
sudo zypper install libayatana-appindicator3-devel
```
